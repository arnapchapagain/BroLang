# BroLang

A "funny" programming language's compiler.

## Intented syntax
Can change overtime

```c#
// This is a comment

num number = 10;
deci decimal = 10.5;

// tf = true/false
tf boolean = true; 

// fr = for real text
fr text = "Hello Bro";

// WACK = null
// try = if
try text is WACK {
    // MF = My Friend
    scream "Empty MF";
}
// again = else if
// cap = is not
again text is WACK {
    scream "Hello Bro";
}
// aight = alright / else
aight {
    scream "Hello Bro";
}

handshake 10 times as i {
    scream i;
}

num add = 10 PLUS 10; // can use + instead
num sub = 10 MINUS 10; // can use - instead
num mul = 10 TIMES 10; // can use * instead
deci div = 10 BY 10; // can use / instead
num power = 10 RAISE 10 // can use ^ instead

fr bro function() {
    yeet "Help me"
}




```
